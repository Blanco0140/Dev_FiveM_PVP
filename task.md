# Roadmap Serveur FiveM PVP Headshot

## Étape 1 : Préparation de l'environnement
* [ ] Installer Visual Studio Code
* [ ] Installer XAMPP ou MariaDB/HeidiSQL

## Étape 2 : Création du Serveur de Base
* [ ] Installer les artefacts FiveM (FXServer)
* [ ] Configurer txAdmin
* [ ] Lancer le serveur et se connecter une première fois

## Étape 3 : La Base de Données & Sauvegardes
* [ ] Créer la base de données et la table des joueurs
* [ ] Script : Initialisation des données (0$, 0 XP)
* [ ] Installation de oxmysql

## Étape 4 : Le Cœur du PVP
* [ ] Script : Headshot = One Shot
* [ ] Système de respawn instantané
* [ ] Désactivation de l'IA GTA (Peds, Trafic, Police)

## Étape 5 : L'Économie & Le HUD
* [ ] Création de l'interface graphique (Kills, Morts, XP, $)
* [ ] Script de récompense de kill (1 kill = 100 XP + Kills +1)

## Étape 6 : Le Starter Pack
* [ ] Intégration de ox_target pour l'interaction PNJ
* [ ] Placement du PNJ
* [ ] Script de don d'armes (Pistolet Mk2 + Cal 50)

## Étape 7 : Les Boutiques (Armes & Balles)
* [ ] Création du menu d'achat avec ox_lib
* [ ] Logique d'achat (vérification de l'argent, prix des balles à 1$)

## Étape 8 : Les Squads & Groupes
* [ ] Menu d'invitation Squad
* [ ] Affichage de l'UI Squad (barres de vie)
* [ ] Désactivation du tir allié (Friendly Fire)

## Étape 9 : Les Arènes et Zones
* [ ] Délimitation des grosses zones de PVP (Rouge, Bleu...)
* [ ] Système d'arènes privées (1v1, 2v2...) et téléportation

## Étape 10 : Boutique VIP & Anti-Cheat
* [ ] Système de Coins
* [ ] Boutique VIP (avantages, véhicules spéciaux)
* [ ] Installation de WaveShield
